# Uploading playbook

This doc is a playbook for simple, repeatable uploads.

## Description template

A talk by _speaker_ at the Seoul Rust Meetup about _quick summary_.

_(Optional) abstract_

Why not join us at our next meetup? See what's coming up at https://seoul.rs/.

## Thumbnail image

Video platforms will normally pick a random thumbnail for you.
We have a nice intro screen though, so we can use it!
After rendering a talk video, just grab `work/intro.png`.

## Other YouTube things

- Make sure to set the recording date. This is buried in the extended options.
- Add an end screen using the outro screen and a basic 1 video + subscribe template.
